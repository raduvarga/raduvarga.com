5 Beta 18 Sep 2021
  • Implemented standard "Save" functionality for presets, replacing the previous autosave functionality. The "Extra" mappings will still autosave.
  • Save plugin mappings per channel.
  • "Assign last touched parameter" feature.
  • Automatically detect virtual ports and fix problems associated with them (like midi settings or mappings being forgotten at restart). Removed "Allow Virtual Midi Ports" setting.
  • Combine Mackie and Midi Mode: it is now possible to map custom controls in Mackie Mode.
  • Added "Mackie" section in Extra where special Mackie functions can be mapped.
  • Added "DAW Mode" for Mackie. This creates a virtual port with the suffix "DAW", which you should use in your DAW instead of the original port.
  • "Program Change" messages are now supported in Midi Mode.
  • Added "Relative" button type.
  • Added "No Value" to Midi Output list.
  • Added "Ref" level toggle for Line Inputs.
  • Added "Channel Strip" to AUX channels.
  • Added LED Feedback for Mackie navigation buttons.
  • Added "Gain Reset" and "Volume Reset" to Preamp and Sends.
  • Fixed plugins being skipped in Mackie Mode.
  • Fixed "Global Mappings" causing issues in Mackie Mode.
  • Fixed somoe rare random crashes that could occur.
