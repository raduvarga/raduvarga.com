Version 5 Beta 18 Sep 2021 Download instructions (MacOs)

Midi updates

  • Implemented standard manual "Save" functionality (Cmd+S or File->Save) for preset files, replacing the previous "autosave" functionality. The "Extra" mappings will still use "autosave" (which are globally saved in extra.uax ).
  • Plugin mappings are now saved per channel, so if a plugin is removed and reinserted later into any slot in the channel, the previous mappings will be loaded. You will have to re-create the your plugin mappings for these changes to take effect.
  • Improved Midi Mapping process: After pressing "Start Mapping", clicking any button or moving a fader in Console will choose the appropriate parameter in the app, so you can now map like in a regular DAW: first move in Console, then move on your midi device.
  • Automatically detect virtual ports and fix problems associated with them (like midi settings or mappings being forgotten at restart). Removed "Allow Virtual Midi Ports" setting.
  • "Program Change" messages are now supported in Midi Mode.
  • Added "Relative" button type.
  • Added "Ref" level toggle for Line Inputs.
  • Added "Channel Strip" to AUX channels.
  • Added "Hide Aux", "Hide Talkback" and "Hide Outputs" in Preferences->Console.
  • Added "Gain Reset" and "Volume Reset" to Preamp and Sends.
  • Allow the selection and removal of multiple parameter mappings in the UI.

Mackie updates

  • Combined Mackie and Midi Modes: It is now possible to create custom midi mappings with devices in Mackie Mode.
  • Added "Mackie" section in the "Extra" Channel, where special Mackie functions can be re-mapped. This allows smaller Mackie devices and also Eucon devices to re-map essential functions like Sends, Plugin, Fader Banks, etc.
  • Added "DAW Mode" for Mackie (default is F1, or can be custom mapped in Extra->Mackie). The app will create a virtual port for every Mackie device with the suffix "DAW", which you should use in your DAW instead of the original port. Then using the "DAW Mode" button, you can quickly switch your Mackie device between Console and your DAW. The following DAWs are currently supported with DAW Mode: Logic, Ableton, FL Studio, Bitwig Studio.
  • Added "Send 2" Assign to Mackie Mode, accesible by pressing the "Send" button twice. In this mode, the Select buttons will select the Send (first, second, etc.) and you will control that selected Send for each Channel.
  • Added LED Feedback for Mackie navigation buttons.
  • Fixed plugins being skipped in Mackie Mode.
  • Fixed "Global Mappings" causing issues in Mackie Mode.

General updates

  • Added "No Value" to Midi Output list.
  • Fixed Window parameter not working correctly using toggle. Also added LED feedback to Window, which is limited to working only when the value is changed via midi (and not via mouse in Console).
  • Fixed some rare random crashes that could occur.
  • Migrated from the old discussion group to a new forum .
